# 🐺 DarkFang

> **Passive Cybersecurity Intelligence Toolkit For Defensive Analysis**

DarkFang Is A **Modern GUI-Based Cybersecurity Tool** Built Using **Python + Tkinter / CustomTkinter**.  
It Focuses On **Safe, Non-Intrusive Intelligence Gathering** While Generating A Clean, Structured **Attack Simulation Report** That Is Fully **Educational And Non-Exploitative**.

---

## 🧠 Core Capabilities

- 🔍 **Passive Target Analysis Only**
- 🛡️ **Defensive Security Insights**
- 🌐 **HTTP Header Inspection**
- 🔐 **TLS Metadata Analysis**
- 📂 **Public Endpoint Probing**
- 📊 **Structured Attack Simulation Reports**
- 🖥️ **Modern Desktop GUI**

---

## 🔍 What It Does

- Sends Standard HTTP Requests (**GET / HEAD**)  
- Checks For Common Security Headers  
- Inspects TLS / SSL Details (HTTPS Targets)  
- Probes Publicly Accessible Endpoints:
  - `/robots.txt`
  - `/.git/HEAD`
  - `/.env`
- Generates High-Level Reports (**No Payloads, No Exploitation**)
